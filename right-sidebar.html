All about buttons.

A button element is an anchor with the "button" class applied. There is a default button style along with an alternative button style.

Buttons elements have class modifiers available to change their size.

<a href="#" class="button small"> Small </a>
<a href="#" class="button"> Normal </a>
<a href="#" class="button large"> Large </a>
<a href="#" class="button xlarge"> Xtra Large </a>

Button elements also have classes that enable alternate button styles.

<a href="#" class="button"> Primary </a>
<a href="#" class="button alt"> Alternative </a>
<a href="#" class="button flat"> Primary Flat </a>
<a href="#" class="button alt flat"> Alternative Flat </a>

What about form elements?

Similar to .button elements, form elements and labels will accept class modifiers to change their size. They have the same .flat class that buttons have and submit inputs have an .alt class.















<input class="small" type="text" placeholder="Small">
<input type="text" placeholder="Default">
<input class="large" type="text" placeholder="Large">
<input class="xlarge" type="text" placeholder="Xtra Large">
